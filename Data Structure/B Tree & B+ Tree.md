# B Tree
* 이진 트리를 개선해 하나의 노드에 여러개의 자료를 배치한 트리 자료구조
* 한 노드에 M개의 자료가 배치되면 M차 B-Tree
* 자동으로 균형을 맞춤
*  데이터베이스, 파일 시스템 등에 사용
* 규칙
  - 노드의 자료 수 N => 자식의 수 N+1 
  - 각 노드의 자료 정렬된 상태
  - Root 노드의 자식 적어도 2개 이상
  - Root 노드를 제외한 모든 노드의 자식 적어도 M/2개
  - 외부 노드 모두 같은 레벨 = 외부 노드로 가는 경로의 길이 모두 동일
  - 입력자료는 중복 X
