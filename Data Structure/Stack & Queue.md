# Stack
* 후입선출(Last In First Out : LIFO) 자료구조
* like 프링글스 <= 나중에 들어간 과자부터 꺼내 먹는 상황
* 시간 복잡도 : O(n)
* 공간 복잡도 : O(n)
* 안드로이드의 액티비티 관리, 함수의 콜 스택, 문자열 역순 출력, 연산자 후위 표기법 등 사용
